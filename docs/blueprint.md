# **App Name**: NoteWave

## Core Features:

- Note Creation: Create and save notes.
- Note Editing: Edit and update existing notes.
- Note Deletion: Delete notes as needed.
- Note Listing: Display all available notes.

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) for a calm and focused environment.
- Background color: Light gray (#F5F5F5), nearly white, for a clean look.
- Accent color: Muted green (#82CA9D) for subtle highlights and actions.
- Body and headline font: 'Inter' for a modern, neutral feel.
- Simple, outlined icons in a consistent style, like those used in Material UI.
- Clean and minimalist layout inspired by Google Notes. Prioritize readability and easy navigation.
- Subtle fade-in animations on note load and when editing notes to give some motion feedback.
